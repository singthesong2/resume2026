# 🤔How Make It?
Canva를 이용해 프로젝트에 쓸 디자인 받고, Figma layer로 정보를 불러와 디테일한 디자인을 수정 및 원하는 내용으로 맞춰 초안을 만들고


그 후, HTML과 CSS를 이용해 모바일 대응 반응형 웹 페이지를 만들었으며, 모든 파일들은 GitHub를 이용해 관리 하였습니다.


<br>


# ✒️Point
- Grid와 Flex를 이용하여 구조 배치
- 미디어쿼리를 이용한 모바일 화면 대응 반응형 구현
- @keyframe을 이용하여 Resume 분위기에 맞는 적절한 속도의 애니메이션 구현
- 터치와 커서, 탭을 생각한 hover구현
- hover의 자연스러움을 위해 transition으로 구체화


<br>


# 🔍Self-Feedback
코드를 하나하나 보면서 문제가 발생했을 경우 어디서 이런 문제가 나타났고, 어떤 부분 때문에 원치않은 결과가 나왔는지 분석은 잘하고 있으며,


꾸준히 복습을 한 결과, 이런 상황에서는 이런걸 쓸 수 있다라고 바로 생각나고 행동에 옮겨 가는건 괜찮은 상황이지만, 예외적으로 더 좋은 코드와


최적화로 더 괜찮게 마크업을 할수 있을거 같지만, 생각이 나지 않아 답답합니다. 예를 들어 clamp을 써 자연스럽게 모바일 대응 반응형을 만든다고 하면


어떤 부분에서 스스로 만족하지 않는 구조가 나올때가 있습니다. 그래서 복습을 더 꾸준히 하면서 좀 더 깊게 연구 해보면서 이 Resume 프로젝트도 


더 좋게 수정해 나가겠습니다.
